Instrucciones
1. En una terminal ejecutar "go run cachipun.go"
2. En otra terminal ejecutar el servidor "py .\servidor_tcp.py"

4. Seguir las intrucciones de la tercera terminal para jugar

Instrucciones Bonus
1. En una terminal ejecutar "go run cachipun.go"
2. En otra terminal ejecutar el servidor "py .\servidor1_tcp.py"
3. En otra terminal ejecutar el servidor "py .\servidor2_tcp.py"
4. En otra terminal ejecutar "py .\cliente1_tcp.py"
5. En otra terminal ejecutar "py .\cliente2_tcp.py"
6. Seguir las intrucciones de juego
7. Para salir, se coloca la opcion 2 en cliente1_tcp.py y luego en cliente2_tcp.py
